## Installation

Requirement nodejs, npm, ganache

```bash
  1. git clone https://github.com/C0DE-SLAYER/kidney_donation_blockchain
  2. cd kidney_donation_blockchain
  3. npm i
  4. open ganache and new workstaion and in the setting select the truffle.config file hit save and restart
  5. npx truffle migrate --reset
  6. npm run dev
  7. In metamask add network and add new network
    - Network name
      - ganache
    - New RPC URL
      - http://127.0.0.1:7545
    - Chain ID
      - 1337
    - Currency symbol
      - ETH
  8. added private key for ganache to metamask 
  9. all set!
```
